Instructions to Operate


Use this simulation to learn how to use the PID Controller to control motor angle

  1. You are given four sliders connected to a text box. Enter the value you want to enter into these text boxes.
  2. For Kp,Ki and Kd, the legal values are -100 to 100 while for required angle it is 0o to 360o
  3. Press `Start Simulation` to begin the simulation and watch how as the motor attains the angle in the camera.
  4. There is also a graph between position and time passed which shows how PID stabilized the error using the values you gave.
  5. The graph comes after you press the Show Graph button on the website.
  6. You need to wait for 15 seconds after running the simulation to give it time to reach the required state.
